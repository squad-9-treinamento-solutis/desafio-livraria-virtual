#### Retorna todos os autores

```http
GET /api/authors
```

#### Altera um autor

```http
PUT /api/authors
```

#### Cria um autor

```http
POST /api/authors
```

#### Retorna um autor

```http
GET /api/authors/{id}
```

#### Deleta um autor

```http
DELETE /api/authors/{id}
```
